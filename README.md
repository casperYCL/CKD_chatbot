# 腎臟衛教機器人（CKD Chatbot）

這是一個腎臟衛教聊天機器人，目的是協助病人或民眾了解慢性腎臟病（CKD）相關的知識與照護資訊。

## 🩺 功能簡介

-  回答常見症狀、飲食建議、治療選項（如血液透析、腹膜透析、腎移植）
-  根據使用者輸入的關鍵字進行互動式回覆
-  具備簡易網頁介面，方便操作與查詢

## 🛠️ 技術架構

- Python 3.9
- Streamlit — 前端介面
- Neo4j — 用於知識圖譜儲存與查詢
- Git — 版本控制
